## Project Overview

This project demonstrates how Prompt Engineering can be used to generate high-converting User Generated Content (UGC) advertisements for real products.

The objective was to design reusable prompts that generate:

* Multiple UGC Hooks
* UGC Ad Scripts
* Call-to-Action (CTA) Variations

The generated content is designed for:

* Instagram Reels
* Short Video Ads
* Social Media Marketing Campaigns

## Prompt Logic

### Hooks Prompt

Designed to generate multiple attention-grabbing UGC hooks using:

* Problem-based hooks
* Curiosity hooks
* Mistake-based hooks
* Before/After hooks
* Trend-style hooks

### UGC Script Prompt

Designed to generate short-form ad scripts following:

Hook → Problem → Solution → Benefit → CTA

### CTA Prompt

Designed to generate:

* Soft CTAs
* Direct CTAs
* Curiosity-based CTAs
* Social-proof CTAs
